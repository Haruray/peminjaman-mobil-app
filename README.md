## Tech Stack

-   Laravel Breeze
-   Vue.js
-   Postgres
-   Visual Studio Code
-   Docker

## Prerequisites

-   PHP v8.1.25 (minimal)
-   Docker

## How to Run

-   create `.docker/postgresqldata` directory
-   run `docker compose up -d` for running postgresql
-   run `composer install` dan `npm install`
-   run `php artisan serve` and `npm run dev`
